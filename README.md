📒 CODSOFT Task 5 - Contact Book

📌 Project Overview

This project is a GUI-based Contact Book developed using Python, CustomTkinter, and JSON as part of the CODSOFT Python Programming Internship.

The application enables users to efficiently manage their contacts by adding, searching, updating, and deleting contact information. Contact details are stored locally in a JSON file, ensuring that data persists even after the application is closed.

---

✨ Features

- ➕ Add new contacts
- 🔍 Search contacts by name or phone number
- ✏️ Update existing contact information
- 🗑️ Delete contacts
- 💾 Automatically save contacts using JSON
- 📋 Display all saved contacts
- 📱 Store name, phone number, email, and address
- 🌙 Modern dark-themed interface using CustomTkinter

---

🛠️ Technologies Used

- Python 3
- CustomTkinter
- Tkinter (Listbox & MessageBox)
- JSON
- OS Module
